核心改进模块

🎯
CBAM 注意力机制
通道注意力 + 空间注意力，增强特征提取能力，提升小目标检测效果
📐
DIoU 损失函数
距离 IoU 损失，优化边界框回归，加速收敛，提高定位精度
🔗
DeepSORT 跟踪
ReID 特征 + 卡尔曼滤波，实现多目标稳定跟踪，保持 ID 一致性
⚡
速度估算
帧间位移计算 + EWMA 平滑，实时估算车辆速度，支持多车道
📊
跨线计数
虚拟检测线 + 方向判别 + 防重复计数，统计车流量
🪶
轻量化设计
深度可分离卷积，模型压缩，参数量仅 4.8M，适配边缘设备
